# VitalShift
A mod that changes your avatar based on your health, allows for visual damage to player if paired with the right avatars.
Last tested on Melonloader 0.6.5, BoneLab patch 6.

## Current Features
- Avatar Selection based on health.
- Disable dying (you still get damage, the last blow just won't kill you)
- Disible automatic healing
